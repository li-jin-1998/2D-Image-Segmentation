# [2D-Image-Segmentation](https://github.com/li-jin-1998/2D-Image-Segmentation)

Personal code for segmentation.

## Usage

export environment
```
conda env export -n lj_py > myenv.yml
```

create environment
```
conda env create -f myenv.yml
```
